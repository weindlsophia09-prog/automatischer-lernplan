# Automatischer Lernplan

## Projektübersicht
Der Automatischer Lernplan ist ein innovatives Tool, das Schülern und Studenten hilft, ihre Lernaktivitäten effizient zu planen und zu organisieren. Das Ziel des Projekts ist es, Lernziele zu setzen, Fortschritte zu verfolgen und personalisierte Lernpläne zu erstellen, um optimal auf Prüfungen und Tests vorbereitet zu sein.

## Funktionen
- **Lernplanerstellung**: Nutzer können individuelle Lernpläne basierend auf ihren Bedürfnissen und Zielen erstellen.
- **Fortschrittsverfolgung**: Verfolgen Sie Ihre Fortschritte beim Lernen, um motiviert zu bleiben.
- **Benachrichtigungen**: Erhalten Sie Erinnerungen für bevorstehende Lernaktivitäten oder Prüfungen.
- **Statistiken**: Analysieren Sie Ihre Lerngewohnheiten und optimieren Sie Ihren Lernprozess.

## Technologies
- **Sprache**: Python
- **Frameworks**: Flask für Backend, React für Frontend
- **Datenbank**: SQLite für die Speicherung der Benutzerdaten

## Installation
1. Klonen Sie das Repository:
   ```bash
   git clone https://github.com/weindlsophia09-prog/automatischer-lernplan.git
   ```
2. Navigieren Sie zum Projektverzeichnis:
   ```bash
   cd automatischer-lernplan
   ```
3. Installieren Sie die erforderlichen Abhängigkeiten:
   ```bash
   pip install -r requirements.txt
   ```
4. Starten Sie die Anwendung:
   ```bash
   python app.py
   ```

## Nutzung
- Melden Sie sich mit Ihren Benutzerdaten an oder erstellen Sie ein neues Benutzerkonto.
- Beginnen Sie mit der Erstellung Ihres persönlichen Lernplans.
